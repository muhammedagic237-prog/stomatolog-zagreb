# Privatna stomatološka ordinacija Marija Rašin dr. stom.

Modern, responsive website for Dr. Marija Rašin's dental clinic in Zagreb.

## Features
- **Modern Design:** Clean, professional aesthetic with premium teal and gold color palette.
- **Responsive Layout:** Fully optimized for desktop, tablet, and mobile devices.
- **Animations:** Smooth scroll reveals, hover effects, and interactive elements.
- **Service Showcase:** Detailed cards for all dental services.
- **Gallery:** Interactive image gallery with lightbox.
- **Contact Form:** Functional contact form with validation.

## Tech Stack
- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (Vanilla ES6+)
- Google Fonts (Playfair Display, Inter)
- Unsplash API (Images)

## How to Run
Simply open `index.html` in any modern web browser.
